Cog

Compile Order Generator for VHDL written in Python

Download this project as a .zip file Download this project as a tar.gz file

Welcome

This project aims to facilitate your design/simulation process by not having to care about compile orders anymore. Simply configure your libraries and optionally your top design file and cog.py will generate a list of compile order.

For a simple demonstration download the files from the link above in either tar.gz or zip format and run the src/simple.py script.

Future features

Currently, only VHDL is supported, but it is my strong intent to support Verilog flavours in the soon future as well.
